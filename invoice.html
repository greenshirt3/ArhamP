Data Backup & Sync

Export your data to sync across devices or create backups. Both systems share the same data.

Select exported JSON file to restore your data
How to Sync Across Devices:
  1. Export Data: Click "Export All Data" on your current device
  2. Save Backup: Save the downloaded file to cloud storage (Google Drive, Dropbox, Email, etc.)
  3. Transfer: On your other device, download the backup file from cloud storage
  4. Import Data: Click "Import Data" and select the downloaded file
  5. Complete: All your customers, invoices, and calculator data will be restored

Tip: Use "Auto Backup" to automatically create backups every 5 minutes
